# Report

This folder contains report-facing materials for the retrospective synthetic EHR post-discharge utilization workflow.

Current report:

- `final_report.md` summarizes the completed data profiling, SQL cohort construction, validation QA, aggregate BI outputs, current results snapshot, and limitations.

The current report is intentionally limited to workflow, validation, and aggregate output findings. It does not include logistic regression or causal interpretation because the modeling milestone has not been completed.

Report materials should use only reproducible outputs generated by the project workflow. No fake findings, real patient data, or patient-level synthetic records should be added.
