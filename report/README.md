# Report

This folder contains report-facing materials for the retrospective synthetic EHR post-discharge utilization workflow.

Current report:

- `final_report.md` summarizes the completed data profiling, SQL cohort construction, validation QA, aggregate BI outputs, descriptive analysis outputs, exploratory logistic regression, current results snapshot, and limitations.

The current report includes exploratory logistic regression results from synthetic data. It does not include causal interpretation or clinical validation.

Report materials should use only reproducible outputs generated by the project workflow. No fake findings, real patient data, or patient-level synthetic records should be added.
