# Report

This folder is reserved for manuscript-style report materials after the SQL cohort, validation, BI summaries, descriptive analysis, and modeling steps are complete.

Planned report materials may include:

- methods summary for the retrospective synthetic EHR workflow
- cohort definition and validation narrative
- Table 1 and descriptive results
- readmission and post-discharge utilization summaries
- model results after the modeling milestone is complete
- limitations and responsible-use language

No fake findings should be added. Report materials should use only reproducible outputs generated by the project workflow.
